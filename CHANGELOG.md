# Changelog

All notable changes to this project will be documented in this file.

This project follows a version-based development approach, where each version introduces one major AI engineering concept.

---

## [1.0.0] - 2026-07-19

### Added

- Initial release of ScholarAI
- Streamlit-based user interface
- PDF upload and text extraction
- LLM integration using OpenRouter
- Prompt management module
- Model-agnostic LLM architecture
- Streamlit Session State for document persistence
- Modular project structure (`src/`)
- Project documentation and GitHub repository

### Notes

This release establishes the foundation for ScholarAI. Future versions will build upon this architecture without rebuilding the project from scratch.
